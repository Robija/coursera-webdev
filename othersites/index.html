CSS rule

h1 {
text-align: center;
}

article > p {
color:blue;
}

This <p> should be blue because the p-tag is a direct descendent of an article-tag

  • Selector: "h1"
  • Declaration "text-align: center;"
  • Property: "text-align"
  • Value: "center"
  • Art of selector : element selector
The "a"-tag is an inline as well as a block element
